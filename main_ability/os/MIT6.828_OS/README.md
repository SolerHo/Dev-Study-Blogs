> 课程中未明确必须具备某些基础。个人认为有对应的基础，可以帮助你更好更快去理解知识点内容。

以下是个人认为重要的内容：
- 编程语言（C语言 / C++）
- 汇编语言
- 计算机组成与体系结构
- Shell
- `csapp`书籍或课程「推荐（个人是看过该内容之后才来看该课程）」

以上内容并不是说你要学会之后才可以看该课程的内容，可以用`边学边做`的方式进行，只不过需要花费的时间比较长。最起码要具有程序语言的基础，可以为你节省不少时间。
